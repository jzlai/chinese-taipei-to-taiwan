# Chinese Taipei to Taiwan Browser Extension

Browser extension for Chrome and Firefox that replaces "Chinese Taipei" with "Taiwan 🇹🇼"

## Installation

### Chrome

1. Download project and unzip
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click `Load unpacked` and select the root folder

### Firefox

1. Download project and unzip
2. Go to `about:debugging#/runtime/this-firefox`
3. Click `Load Temporary Add-on`
4. Select any file in the root folder (e.g., `manifest.json`)

**Note**: In Firefox, the extension will be removed when you restart the browser. For permanent installation, you need to sign and publish it to addons.mozilla.org.
